    Project Ideas:

    My dream project at the moment is to build a help-to-study app for students especially college and university students. This idea doesn’t seem to be new. As the matter of fact, learning apps have existed for a very long time. However, I want to put my own twist on the app in order to make it more useful to both students and teachers.

    I have been using learning app for a few years now. Most of the apps, although very useful in various of scenarios such as provide key detail, important information, tips/tricks to study and solutions, still have 1 major flaw in common: the connection. “The connection” here is not about Wi-Fi or cellular data, it is about the connection between student and the lecture/lesson. Study face-to-face has always been more appealing than study entirely through a screen. You don’t get the excitement from listening to an amazing lecturer, you miss all the fun talking with your friends in class, you don’t have the explanation, guidance and help from the teachers. So, instead of trying to make an improved self-study app, I want to make an app that hep to improve the quality of the lessons in class.

    The app that I want to build is going to be a mobile device app. By doing this, the app will be very accessible to all the students and teachers. In best case scenario, the app should be pre-installed on every study device in school/university. The application is going to be a multi-function app. First, the app has to have all the lecture file and recorded video of the lecture so that students can revise or study at home in case the student miss the lesson, similar to other study app. Not only the lecture, homework should also be put on this app. For the sake of simpler assessing students’ work, homework should be written in multiple choice format and automatically assess by the app. By doing so, it encourages students to spend their time revise the lesson and practice by doing homework. Besides, this method can save the teachers a lot of time checking students’ work and helping students who not fully understand the lecture. Another problem needs to address is the lack of understanding of an ongoing class session. A lot of students don’t fully understand the lecture for various reason. For instance, the language barrier can prevent students from following the lesson or sometimes the reason can simply be lack of attention. I believe this problem happens due to the so little interaction happen between teacher and student especially when there are 100 students in a class. Students who can’t follow the lecture tend to stay silent instead of asking question because of shyness. To tackle this problem, I want to implement the real time feedback feature into the app. During the lecture, after a section, students have to answer a set of questions relate to that section or give a brief summary of that section. The answers will be checked either by the app or the lecturer and his/her assistant. This will establish a lot of students’ interaction and participation in the lecture and also help teachers identify which student need help, hence improve the lecture quality. Another feature I would love to include in the app is course recommendation. The app will analyze the students’ major, their scores in every course, identify their weakness and strength in order to give suggestion in course selection. Finally, I want the app to report the real time busyness of each location around the campus. This last feature doesn’t directly improve the study quality; however, I believe this will become very handy in many situations. For example, students by using this feature can know how many people inside the library before they come to the library. If the library is too crowded, they can use online library instead or come to the library in another time. This can simply be done by using some small devices like Raspberry Pi to log MAC address of Wi-fi and Bluetooth device nearby similar to the way they do in shopping mall or airport.

    To build this app, first you need to have some coding program, something like Swift, Xcode or Android Studio, … You also need to design the user interface with Adobe Photoshop, Adobe Experience Design, … To store user data, lectures, marks, ... you also need to have a server. And finally, some device to collect data around campus such as Raspberry Pi.

    To be able to build the app, I think I will need to learn a few programming languages such as C, C++, Java, Python, Swift, … Besides, it also will be essential to learn HTML and JavaScript for web designing and server operating. Additionally, I also need to able to use Adobe Photoshop, Experience Design, Illustrator in order to design user interface for the app.

    I would say this project although is a “dream project”, it is actually very practical. If the project is successful, I believe this app will drastically improve learning quality and students’ interest. It will help millions of students. However, it will not be easy. In order for it to work out, the project need a huge support from schools, universities, teachers and students. I still have a long way to go but hopefully one day I will make it.
